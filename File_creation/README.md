# Shell File Task

In this task, I worked with basic Linux file operations using shell commands.

I started by creating a directory named `my_folder` and moved inside it. Then I created two files, `my_file.txt` and `another_file.txt`, and added some sample text using echo.

After that, I appended the content of `another_file.txt` into `my_file.txt` and verified the output.

Next, I created 20 text files using a single command and renamed the first 5 files from `.txt` to `.yml`.

Finally, I listed the most recently created files using sorting commands in the terminal.

All the outputs for each step are included in the screenshots folder.
