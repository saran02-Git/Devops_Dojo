# Jenkins Setup on AWS EC2

## Objective

Launch Jenkins on AWS EC2 and explore creating users and projects.

## Tools Used

* AWS EC2
* Jenkins

## Steps Performed

1. Launched EC2 instance and installed Jenkins.
2. Accessed Jenkins using public IP.
3. Created a new user in Jenkins.
4. Created a freestyle project.
5. Ran a build and verified output.

## Output

* Jenkins is running successfully on EC2.
* User created successfully.
* Project created and build executed successfully.

## Conclusion

Successfully completed Jenkins setup and basic operations.
