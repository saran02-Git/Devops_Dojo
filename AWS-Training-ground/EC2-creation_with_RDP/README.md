# AWS Task 1 – Windows EC2 Setup

## Overview

This is a simple hands-on task to understand how a Windows virtual machine can be created and accessed in AWS.
The goal was to launch a Windows EC2 instance, connect to it using Remote Desktop (RDP), and verify the system details from inside the machine.

---

## What I Did

* Launched a **Windows Server 2022 EC2 instance** in AWS
* Configured **RDP access (port 3389)**
* Connected to the instance using **Remote Desktop**
* Opened Command Prompt inside the VM
* Retrieved system details using a basic command

---

## Command Used

```
systeminfo
```

## Key Learning

This task is a basic introduction to working with cloud virtual machines.
It shows how we can create and access a remote Windows system from anywhere, which is a core concept in cloud and DevOps environments.
